# shiny_modular
Este es el ejemplo de la construcción de una aplicación de shiny con módulos. El tutorial se encuentra en el **blog de ixpantia** y se describen los pasos para entender cómo funcionan los módulos en shiny.

Recomendamos descargar este código para seguirlo junto con el tutorial y hacer cambios (hasta que se rompa) para lograr comprender mejor cómo se construyen los módulos y cómo logramos darles interactividad.
